Terms


Oppia is an open source project, and its code is released under an Apache 2.0 license. By submitting contributions to either the code repository or the Oppia demo server, you agree that such contributions will also fall under the Apache 2.0 license. Please don't submit stuff that you don't have the rights to submit or that is incompatibly licensed.

Please be aware that Oppia is being developed actively, and that the Oppia test server can go down at any time. In addition, content on the test server site may be deleted without notice until Oppia moves into beta (non-preview) mode. We suggest that, in the meantime, you take backups of your explorations by downloading a YAML file from the exploration editor page, and keeping a copy of any images that you upload to the server. (We are hoping to get into a more stable state soon.)
